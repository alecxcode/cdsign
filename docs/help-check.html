Check document digital signature: how to use the app

Introduction

A digital signature is a mathematical scheme for verifying the authenticity of electronic documents. With this app it uses secret and public keys, and detached digital signature file.
With secret key you can sign an electronic document, secret key should not be transferred to any other person. Public key should be given to another person in order they can verify your signature and document integrity.
When you sign a file with a digital signature a hash is being generated, which then modified by cryptographic algorithms. This modified hash is the digital signature of a specific electronic document.
All this is essential in order to verify (using public key and that hash) that a a specific electronic document signed by a person specified in that public key and not modified during transmission (integrity check).
Although, this program does not utilize any certificate authority (an entity that issues digital certificates).
This application itself does not have any cryptographic algorithms, for this purpose it utilizes another program - GPG. This app is basically simplified graphical user interface for GPG.
In order to check a digital signature you need from a signing person: (1) a signed file itself, and (2) a public key of that person (if not received earlier), and (3) a digital signature file related to a signed file - all of them created with the application Digitally sign a document.

Import a public key of a digital signature

Import of a public key is made during the digital signature check process, if you have selected a key file. You need to import this key if you did not import in before. Otherwise, there is no sense to import the key.
In order to choose a public key file press Import public key... button. That kind of file usually has an extension of .key. It will be imported while checking digital signature, not other actions required.
The imported public key will be stored in a keyring of the GPG program.

Check a digital signature of a document

In order to check digital signature of a file and that file integrity:
  1. Launch the application
  2. Press the button Document (or contract) file...
  3. Select a signed file
  4. Press the button Signature file for this document... A digital signature file usually has the same name as a document file with an extension of .asc
  5. Select a signature file
  6. If necessary select a public key file to import
  7. Press the button Check digital signature
  8. The application should display the result

Application settings (for advanced users)

The settings are: GPG executable name.
The settings are stored in: user_home_directory/.edm/cdsign-check.ini
